""""
    A robots.txt file is a text file placed in a website's root directory that instructs search engine
     crawlers (bots) which pages or files they can or cannot request. It manages crawl budget, 
     prevents crawling of duplicate or sensitive content, and is located at yourdomain.com/robots.txt. 

    Syntax:
        User-agent: Defines which bot the rule applies to (e.g., * for all, Googlebot for Google).
        Disallow: Tells the bot not to crawl a specific page or directory.
        Allow: Overrides a Disallow rule for a specific subdirectory.
        Sitemap: Specifies the location of the XML sitemap.

    (source : wikipedia)
"""
